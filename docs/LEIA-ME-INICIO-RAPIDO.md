## 🔧 Comandos Rápidos

### Verificar Status
```bash
# Backend
lsof -i :5000

# Frontend  
lsof -i :3000

# Database
docker compose ps
```

### Reiniciar Serviços
```bash
# Backend
cd backend && flask run --reload

# Frontend
cd frontend && npm start
```
